HTML

簡介

HTML全名為HyperText Markup Language，是一種標示語言。其中的元素是建構網站的基石，可與CSS、JavaScript搭配在網頁瀏覽器上產生各種不同的效果。
本網頁主要參考HTML Tutorial，並使用線上轉譯工具協助編寫。

  • 申請上傳空間

    • 為了讓HTML在網路上運作，我們需要一個儲存HTML檔案的空間，這邊我利用的是名叫GitHub的空間，使用方法如下:

    1. 進入網站後申請帳號
    2. 點選Start a project，建立專案(repository)，進入後輸入專案名稱，點選Create respository
    3. 將Code選單切換至Settings
    4. 尋找區塊 GitHub Pages中的Theme chooser點選choose a theme
    5. 任意挑一個theme後，點Select theme
    6. 將完成的HTML檔案上傳到Code分頁即完成上傳的作業
    7. 連結的網址的格式為https://帳號名.github.io/專案名/檔名

  • 格式

    • 要使用HTML中的功能需要借助tag來標示，使用方法如下:
          <starttag> 內容 <endtag>


    通常starttag和endtag會成對出現，後者會比前者多一個forward slash「\」。如<p>...</p>

  • 結構

    • HTML的基本架構如下表，其中<head> tag裡面可以控制頁面的標題，而平常網頁顯示的內容主要編寫在在<body> tag裡面。
    <html>
    <head>
    <title>Page title</title>
    </head>
    <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
    <p>This is another paragraph.</p>
    </body>
    </html>
