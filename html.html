HTML

簡介

HTML全名為HyperText Markup Language，是一種標示語言。其中的元素是建構網站的基石，可與CSS、JavaScript搭配在網頁瀏覽器上產生各種不同的效果。

  • 編輯工具與參考資料

    • 編寫HTML只要打開記事本按照HTML的編寫格式編寫就行了，不過在儲存檔案的時候，需要將副檔名從原本預設的.txt，更改為.html。
    本網頁主要參考HTML Tutorial，並使用「線上轉譯工具」協助編寫，網頁中使用到的顏色則可以使用Color Picker來調配。

  • 申請上傳空間

    • 為了讓HTML在網路上運作，我們需要一個儲存HTML檔案的空間，這邊我利用的是名叫GitHub的空間，使用方法如下:

    1. 進入網站後申請帳號
    2. 點選Start a project，建立專案(repository)，進入後輸入專案名稱，點選Create respository
    3. 將Code選單切換至Settings
    4. 尋找區塊 GitHub Pages中的Theme chooser點選choose a theme
    5. 任意挑一個theme後，點Select theme
    6. 將完成的HTML檔案上傳到Code選單中即完成上傳的作業
    7. 連結的網址的格式為https://帳號名.github.io/專案名/檔名

  • 格式

    • 要使用HTML中的功能需要借助tag來標示，使用方法如下:
          <starttag> 內容 <endtag>

    通常starttag和endtag會成對出現，後者會比前者多一個forward slash「\」。如<p>...</p>

  • 結構

    • HTML的基本架構如下表，其中<head>裡面可以控制頁面的標題或整個網頁的統一格式，而平常網頁顯示的內容主要編寫在在<body>裡面。
    <html>
    <head>
    <title>Page title</title>
    </head>
    <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
    <p>This is another paragraph.</p>
    </body>
    </html>

<head>

<head>是一個用來形容整個網頁的區塊，即整個網頁的主要編寫資訊都會包含在此區塊。其中包含<title>, <link>, <style>, <meta>, <script>, 和 <base>等tags可供使用。

  • title

    • 可設定網頁的標題

  • link

    • 用來控制網頁內與外部資源的關連，例如使用:
    <link rel="icon" href="圖片網址" type="image/x-icon">
    就能設定標題的圖案

  • style

    • 用來定義網頁中各個tag所使用的style，可結合css中的class一同使用

<body>

如前所述，編寫網頁的內容主要在<body>這個tag中。<body>本身也有一些功能可供設定
  • bgcolor設定網頁背景顏色
  • background設定背景圖案

<p>

<p>指的是paragraph，其中可使用title這個元素設定整個斷落的標題。

<a>

  • href設定超連結
  • name設定名稱，用於超連結時加#
  • style="text-decoration:none" target="_blank"
